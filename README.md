Interactive Spanish Number Quiz
===============================

This document describes an HTML-based interactive quiz application designed to teach Spanish numbers through a user-friendly interface.

Features
--------

### Structure & Styling

-   **Theme:** Dark-themed user interface with a modern look.

-   **Typography:** Uses Google Fonts for a sleek appearance.

-   **Layout:** Centralized and responsive design to fit various screen sizes.

### Functionality

-   **Mode Switching:**

    -   **Sıralı (Ordered):** Users can sequentially navigate through numbers.

    -   **Karışık (Mixed):** Numbers are presented in a random order.

-   **Number Navigation:**

    -   **Jump to Number:** Direct input allows jumping to any number between 1 and 5000.

-   **Progress Tracking:**

    -   **Stats Display:** Shows current, seen, and remaining numbers.

    -   **Progress Bar:** Visual representation of quiz progress.

-   **Answer Reveal:**

    -   Users click to reveal the Spanish translation of the displayed number.

-   **Speech Synthesis:**

    -   **Voice Output:** Spanish numbers can be read aloud.

    -   **Speed Control:** Allows adjustment of speech rate.

-   **Keyboard Interactions:**

    -   Arrow keys and space bar for navigation and revealing answers.

    -   'S' key initiates number reading.

### JavaScript Logic

-   **Conversion Functions:** Converts numeric values to Spanish text representation.

-   **State Management:** Tracks current number, seen numbers, and mode (ordered or mixed).

-   **Speech Synthesis:** Utilizes the Web Speech API to read out numbers in Spanish.

Conclusion
----------

This application offers an engaging and educational experience, combining visual, interactive, and auditory elements to facilitate learning Spanish numbers.
